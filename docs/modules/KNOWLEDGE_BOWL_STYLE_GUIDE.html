Design Philosophy

This style guide defines an accessible, emotionally engaging color system for the Knowledge Bowl module in UnaMentis. The design works for users with colorblindness without requiring a special accessibility mode.

üéØ
Key Principle: Never Color Alone
Every status and state uses color + icon + text label. This ensures information is accessible to everyone, including the 8% of males with red-green colorblindness.
üîÑ
Accessible Substitutions
Teal replaces green for success states. Magenta replaces red for urgency/errors. Both are distinguishable for all common forms of colorblindness.

12-Domain Color Palette

Each academic domain has a distinct color that works in isolation and as part of visualizations like radar charts. Colors differentiate through both hue AND luminance for colorblind accessibility.

Constructive Framing
We use "Focus Area" instead of "Weakness" to encourage rather than discourage. The magenta color signals attention without the negative connotations of red.

Key Insight
Notice how each status remains distinct even under simulated colorblindness. The magenta (leftmost) is always distinguishable from the teal (rightmost) because they differ in both hue AND luminance.
