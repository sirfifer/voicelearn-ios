# Development Tools

This section documents the development tools used in the UnaMentis project.

## Code Review

- [[CodeRabbit]] - AI-powered code review for pull requests

## CI/CD

- [[GitHub-Actions]] - Continuous integration and deployment

## Development Environment

- **Xcode** - iOS development IDE
- **VS Code** - Server and web development
- **Claude Code** - AI-assisted development

## MCP Servers

The project uses Model Context Protocol (MCP) servers for enhanced AI tooling:

- **XcodeBuildMCP** - Xcode build automation
- **ios-simulator** - iOS Simulator control

## Linting and Formatting

| Tool | Purpose |
|------|---------|
| SwiftLint | Swift code linting |
| SwiftFormat | Swift code formatting |
| ESLint | TypeScript/JavaScript linting |
| Prettier | Web code formatting |
| Ruff | Python linting |
| Black | Python formatting |

## Testing

- **XCTest** - iOS unit and integration tests
- **pytest** - Python server tests
- **Jest** - JavaScript/TypeScript tests
- **Latency Harness** - Performance testing
