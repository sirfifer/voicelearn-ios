# Code of Conduct

## Our Commitment

VoiceLearn is committed to providing a welcoming and inclusive environment for everyone. We expect all participants to treat each other with respect and professionalism.

## Expected Behavior

- Be respectful and considerate in all interactions
- Welcome newcomers and help them get started
- Accept constructive feedback gracefully
- Focus on what is best for the community and the project
- Show empathy towards other community members

## Reporting Issues

If you experience or witness behavior that violates this code of conduct, please contact the maintainer at richard@amerman.net. All reports will be handled confidentially.

## Enforcement

Maintainers have the right and responsibility to remove, edit, or reject contributions that do not align with this Code of Conduct, and to temporarily or permanently ban contributors for behaviors they deem inappropriate.
